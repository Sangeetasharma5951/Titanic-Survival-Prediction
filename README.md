# Titanic-Survival-Prediction
Titanic Survival Prediction is a beginner-friendly machine learning project that predicts whether a passenger survived the Titanic disaster based on historical passenger data.

</br>The dataset includes features such as Age, Gender, Passenger Class (Pclass), Fare, Cabin, Embarked Port, SibSp (siblings/spouses aboard), Parch (parents/children aboard), and other passenger details. Using these features, a classification model is trained to predict the Survival status of each passenger.

This project demonstrates the complete machine learning workflow, including:
</br>Data collection and exploration
</br>Data cleaning and preprocessing
</br>Exploratory Data Analysis (EDA)
</br>Feature engineering
</br>Model training and evaluation
</br>Prediction of passenger survival

</br>The goal of this project is to understand how different passenger attributes influenced survival and to build an accurate predictive model using machine learning techniques.
